    Frequently Asked Questions

    Do I need programming experience for this workshop? +
    Basic familiarity with JavaScript or Python is helpful but not required. We'll provide code examples and walk through each step during the live-coding sessions. The focus is on learning through hands-on practice.
    What if I don't have a Google Earth Engine account? +
    We'll help you set up access during the workshop. Make sure you have a Google account ready, and we'll guide you through the Earth Engine registration process step-by-step.
    Can I access the materials after the workshop? +
    Yes! All code examples, tutorials, and datasets will remain available through the NEON Data Portal and our GitHub repository for continued learning and application to your research.
    What ecological applications will we cover? +
    Focus areas include vegetation classification, biodiversity monitoring, ecosystem health assessment, and spectral analysis using NEON's comprehensive airborne datasets across major US ecosystems.
    What makes NEON data special for ecological research? +
    NEON provides standardized, quality-assured data across 81 sites representing major US ecosystems, with 2-10 years of temporal coverage and integration with extensive ground-based measurements for comprehensive ecological analysis.
